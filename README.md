# DanRStyle

**DanRStyle** is a custom R code formatting style designed by Dan Manter.  
It provides a consistent, readable, and reviewer‑friendly formatting workflow that includes:

- pipe‑at‑start formatting  
- tidyverse‑style indentation  
- soft alignment of `=`  
- consistent operator spacing  
- aligned and wrapped comments  
- multi‑line argument breaking  
- git‑aware styling (only restyles changed lines)  
- an RStudio Addin for one‑click styling  

This package extends the `styler` ecosystem with Dan’s preferred visual grammar for ecological and statistical workflows.
